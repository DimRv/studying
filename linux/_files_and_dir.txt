Каталоги:
~ - каталог текущего пользователя
/ - корень, вся файловая иерархия
/bin - основные утилиты
/boot - загрузочные файлы
/dev - файлы устройств
/etc - системные конфигурационные файлы
/home - домашние каталоги пользователей
/lib - основные библиотеки для работы программ из /bin
/media - монтированные внешние носители
/mnt - временно монтируемые файловые системы
/opt - дополнительное программное обеспечение
/proc - состояние ядра ОС и запущенных процессов
/root - домашний каталог root пользователя
/sbin - основные системные программы
/srv - данные о сервисах предоставляемых системой
/tmp - временные файлы
/usr - пользовательские приложения и утилиты в многопользовательском режиме
/var - изменяемые файлы

.file_name - скрытый файл

Основные типы файлов:
- файлы (-)
- каталоги (тот же файл - список относящихся к нему файлов) (d)
- ссылки. Символьная ссылка содержит инфо о другом файле. Жесткая ссылка - еще одно имя файла. (l)
- блочные устройства (b) - устройства \dev оперирующее блоками данных от 512байт
- символьные устройства (с) - устройства \dev передающие один символ в промежуток времени
- именованный канал pipe (p) - служит для связи процессов между собой
- сокеты (s) - средство межпрограммного сетевого взаимодействия

Наиболее популярные файловые системы: ext4 и xfs.
Файловые системы оперируют индексными дескрипторами, которые хранятся в таблице и уникальным номером
Номер ноды на файл можно посмотреть ls -i file_path
Inode - индексный дескриптор

Основные команды для работы с файлами:

pwd - текущая директория
cd - смена директории
mkdir - создание директории
ln - создание ссылки
touch - создание пустых файлов
mknod - создание блочных и символьных устройств в /dev
ls - содержимое директории
dir - содержимое директории
file - определяет тип файла
stat - подробная информация о файле
cp - копирование
mv - перемещение
rmdir - удаление каталога
rm - удаление
cat - содержимое файла
less - чтение файла
more - чтение файла
head - первые строки файла
tail - последние строки файла
lsblk - список блочных устройств
blkid - инфо о блочных устройствах
findmnt - список смонтированных файловых систем
df (disk free) - отчет об использовании фс
du (disk usage) - объем использованного места на диске
mount - монтирование устройства - необходимо перед использованием
unmount - отмонтирование
tar - работа с архивами
gzip - сжатие архива
updatedb - обновление БД для поиска файлов
locate - поиск файлов
whereis - поиск исполняемых файлов, исходные кода
find - поиск с помощью регулярных выражений
